MELPA
This package is available from MELPA, so if you want to install it and have added MELPA to your package archives, you can run M-x package-install RET org-noter

Manual installation
You can also install it manually, using package.el.

Download org-noter.el
Open it
Run M-x package-install-from-buffer RET
